1. Скачать датасет по ссылке из файла preprocessing/links.txt
2. Закинуть его в preprocessing
3. В data_preprocessing.ipynb есть 2 скрипта. Первый - создает выборки со всем фаршем (удаление выбросов, создание признаков и.т.д), второй - просто заполнение пропущенных и масштабирование (для проверки того, не стал ли датасет хуже в результате подготовки :) )
